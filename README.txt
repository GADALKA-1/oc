## Выполненные модули

### Модуль 1 — База данных
- Создана БД SQLite в 3НФ со ссылочной целостностью
- Все справочники, таблицы товаров, заказов, пользователей и ролей
- Предзаполненные тестовые данные

### Модуль 2 — Приложение и авторизация
- Окно входа с проверкой логина/пароля из БД
- Вход как гость (просмотр без фильтров)
- Роли: Гость, Авторизованный клиент, Менеджер, Администратор
- ФИО пользователя в правом верхнем углу
- Кнопка «Выход» на главный экран
- Список товаров с макетом: фото + описание + скидка
- Подсветка: скидка >15% → #2E8B57, нет на складе → голубой, снижена цена → перечёркнута красным

### Модуль 3 — CRUD товаров
- Фильтрация по поставщику (выпадающий список, «Все поставщики»)
- Сортировка по количеству на складе (возрастание/убывание)
- Поиск по всем текстовым полям в реальном времени
- Добавление/редактирование товара (только администратор)
- Загрузка фото с ограничением 300×200 пикселей
- Удаление старого фото при замене
- Невозможность удалить товар, присутствующий в заказе
- Валидация: цена и количество ≥ 0

### Модуль 4 — Заказы
- Просмотр заказов для менеджера и администратора
- Макет: артикул, статус, адрес ПВЗ, даты, дата доставки
- Добавление/редактирование/удаление заказов (администратор)
